About

Posted on 2020-11-27

  • This EFI contains EFI file(s) from Daliansky and some DSDT patch and SSDT patch that enable I2C touchpad and some audio inject for ALC256 on Acer Aspire A514-52-37H1.

  • This will let you to install and run macOS Catalina on Acer Aspire A514-52-37H1 with this EFI.

Tested version :

  • macOS Version : Catalina 10.15.7
  • Clover Version : v2.5k r5120
